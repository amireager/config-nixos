#  راهنمای کلیدهای میانبر دسکتاپ

> کلیدهای میانبر Niri + DMS + ابزارهای سیستمی

---

##  Niri — مدیریت پنجره‌ها

### جابجایی بین پنجره‌ها

| کلید | کار |
|---|---|
| `Mod + H` | پنجره سمت چپ |
| `Mod + L` | پنجره سمت راست |
| `Mod + K` | پنجره بالا |
| `Mod + J` | پنجره پایین |
| `Mod + ←/→/↑/↓` | همان بالا (جهت‌دار) |

### مدیریت پنجره

| کلید | کار |
|---|---|
| `Mod + Q` | بستن پنجره |
| `Mod + Shift + Q` | خروج از برنامه |
| `Mod + F` | بیشینه کردن ستون |
| `Mod + Alt + F` | بیشینه کردن پنجره |
| `Mod + Shift + F` | تمام صفحه |
| `Mod + Ctrl + F` | تمام صفحه پنجره‌ای |
| `Mod + T` | شناور/کاشی کردن |
| `Mod + C` | وسط‌چین کردن ستون |
| `Mod + R` | تغییر اندازه پیش‌فرض |

### تغییر اندازه

| کلید | کار |
|---|---|
| `Mod + -` | کاهش عرض ۱۰٪ |
| `Mod + =` | افزایش عرض ۱۰٪ |
| `Mod + Ctrl + ←/→` | جابجایی ستون |
| `Mod + Ctrl + ↑/↓` | جابجایی پنجره |
| `Mod + Shift + H/L` | جابجایی ستون چپ/راست |
| `Mod + Shift + K/J` | جابجایی پنجره بالا/پایین |

### فضای کاری (Workspace)

| کلید | کار |
|---|---|
| `Mod + 1-5` | رفتن به فضای کاری |
| `Mod + Shift + 1-5` | انتقال پنجره به فضای کاری |
| `Mod + Tab` | نمای کلی (Overview) |
| `Mod + Shift + Tab` | فضای کاری قبلی |
| `Mod + WheelDown/Up` | فضای کاری بعدی/قبلی |

---

##  DMS — پوسته دسکتاپ

### لانچر و کنترل

| کلید | کار |
|---|---|
| `Mod + Space` | باز کردن لانچر برنامه‌ها |
| `Mod + S` | سایدبار (وایفای، بلوتوث، صدا، ...) |
| `Mod + ,` | تنظیمات DMS |

### صدا و روشنایی

| کلید | کار |
|---|---|
| `Volume Up` | افزایش صدا |
| `Volume Down` | کاهش صدا |
| `Volume Mute` | بی‌صدا کردن |
| `Brightness Up` | افزایش روشنایی |
| `Brightness Down` | کاهش روشنایی |

---

##  برنامه‌ها

| کلید | برنامه |
|---|---|
| `Mod + Enter` | ترمینال (Kitty) |
| `Mod + /` | ترمینال سریع (شناور) |
| `Mod + B` | مرورگر (Zen) |
| `Mod + E` | مدیر فایل (Thunar) |
| `Mod + Shift + E` | Yazi (TUI) |

##  عکس صفحه

| کلید | کار |
|---|---|
| `Mod + P` | عکس از ناحیه انتخاب |
| `Mod + Shift + P` | عکس از کل صفحه |
| `Mod + Alt + P` | عکس از پنجره فعلی |

##  میانبرهای سیستمی (Fish Shell)

### مدیریت سیستم

| دستور | کار |
|---|---|
| `nrs` | نوسازی سیستم (nh os switch) |
| `nrt` | تست سیستم (nh os test) |
| `nrb` | ساخت سیستم (nh os build) |
| `nrsv` | نوسازی با خروجی کامل |
| `hms` | نوسازی Home Manager |
| `sw` | میانبر برای `nh os switch` |

### فایل و پوشه

| دستور | کار |
|---|---|
| `ls` | لیست فایل‌ها (eza) |
| `ll` | لیست بلند |
| `la` | شامل مخفی |
| `lt` | درخت |
| `cat` | خواندن فایل (bat) |
| `find` | جستجوی فایل (fd) |
| `grep` | جستجوی متن (ripgrep) |
| `top` | مانیتور سیستم (btop) |

### گیت

| دستور | کار |
|---|---|
| `gst` | وضعیت گیت |
| `gaa` | افزودن همه |
| `gl` | تاریخچه |
| `lg` | lazygit |

### جستجو و تاریخچه

| دستور | کار |
|---|---|
| `Ctrl+R` | جستجوی تاریخچه (atuin) |
| `z` | پرش هوشمند (zoxide) |
| `zi` | پرش تعاملی |
| `tldr` | راهنمای سریع دستورات |
